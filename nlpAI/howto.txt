requirements
------------
- a local directory for code and local testing
- a service on google appengine for remote deployment


virtual environment
===================
> virtualenv env  #one time at init
> source env/bin/activate

deploy to existing appengine
============================
pip freeze > requirements.txt
gcloud app deploy --project postcard-251716
gcloud app browse --project=postcard-251716
browser> https://postcard-251716.appspot.com/

local simulation python 3
================
source env/bin/activate
export FLASK_APP=main
flask run
127.0.0.1:5000 #in chrome
